# Xam.Bin.Controls
Custom controls for xamarin forms

Available on NuGet: https://www.nuget.org/packages/Xam.Bin.Controls/

Install into your PCL project and Client projects.

List of controls

- BinEntry
- BinPicker
- BinSegemented Control

Add a namespace
```
using Xam.Bin.Controls.Controls;

```
